Understanding Functions in C++

Easy Problems

(1) Write a function that takes no parameters and prints "Hello, World!".

(2) Write a function that takes two integers as parameters and returns their sum.

(3) Write a function that takes an integer and returns its square.

(4) Write a function that takes a floating-point number and returns its absolute value.

(5) Write a function that takes a character and returns its ASCII value.

(6) Write a function that takes an array of integers and its size as parameters, and prints all the elements.

(7) Write a function that takes an integer array and its size, and returns the maximum value.

(8) Write a function that takes a string and returns its length.

(9) Write a function that takes a string and a character, and counts the occurrences of the character in the string.

(10) Write a function that takes two strings and returns their concatenation.

(11) Write a function that takes a number and returns true if it is even, and false otherwise.

(12) Write a function that takes a number and returns true if it is a prime number, and false otherwise.

(13) Write a function that takes an integer and returns the factorial of the number.

(14) Write a function that takes two numbers and returns their greatest common divisor (GCD).

(15) Write a function that takes a number and returns its Fibonacci number.
