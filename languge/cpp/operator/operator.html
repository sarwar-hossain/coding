Understanding Operators in C++

Easy Problems

(1) Write a program to demonstrate the use of arithmetic operators (+, -, *, /, %).

(2) Write a program to demonstrate the use of relational operators (==, !=, >, <, >=, <=).

(3) Write a program to demonstrate the use of logical operators (&&, ||, !).

(4) Write a program to demonstrate the use of assignment operators (=, +=, -=, *=, /=, %=).

(5) Write a program to demonstrate the use of bitwise operators (&, |, ^, ~, <<, >>).

(6) Write a program to demonstrate the use of increment (++) and decrement (--) operators.

(7) Write a program to demonstrate the use of conditional (ternary) operator (?:).

(8) Write a program to demonstrate the use of the comma operator (`,`).

(9) Write a program to demonstrate the use of the member access operator (`.`) and pointer-to-member access operator (`->`).

(10) Write a program to demonstrate the use of the scope resolution operator (`::`).

(11) Write a program to demonstrate the use of the sizeof operator to determine the size of different data types.

(12) Write a program to demonstrate the use of the type casting operators (static_cast, dynamic_cast, const_cast, reinterpret_cast).

(13) Write a program to demonstrate the use of the address-of operator (`&`) and dereference operator (`*`).

(14) Write a program to demonstrate the use of the new and delete operators for dynamic memory allocation.

(15) Write a program to demonstrate the use of the function call operator (`()`).
