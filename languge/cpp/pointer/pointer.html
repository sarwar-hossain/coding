Understanding Pointers in C++

Easy Problems

(1) Write a program to declare an integer variable, a pointer to the integer, and print the value of the integer using the pointer.

(2) Write a program to dynamically allocate memory for an array of integers using a pointer and initialize the array with values.

(3) Write a program to demonstrate pointer arithmetic by creating an array of integers and using pointers to iterate through the array.

(4) Write a program to swap two integers using pointers and a function.

(5) Write a program to find the length of a string using a pointer.

(6) Write a program to demonstrate the use of a pointer to a pointer. Create a pointer to an integer, then create another pointer that points to the first pointer.

(7) Write a program to pass an array to a function using pointers and modify the array elements inside the function.

(8) Write a program to allocate memory for a 2D array dynamically using pointers and display the elements of the array.

(9) Write a program to use a pointer to store and manipulate a floating-point number.

(10) Write a program to create a structure with some members and use pointers to access and modify the structure members.

(11) Write a program to demonstrate the use of the `nullptr` keyword by initializing a pointer to `nullptr` and checking its value.

(12) Write a program to implement a basic pointer-based linked list with insert and display functions.

(13) Write a program to use a pointer to pass an integer by reference to a function and modify its value.

(14) Write a program to demonstrate the difference between `&` (address-of) operator and `*` (dereference) operator.

(15) Write a program to dynamically allocate memory for a string using a pointer and perform operations like concatenation and comparison.
