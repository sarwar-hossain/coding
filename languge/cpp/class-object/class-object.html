Understanding Classes and Objects in C++

Easy Problems

(1) Write a class to represent a simple rectangle with width and height. Include methods to calculate the area and perimeter.

(2) Write a class to represent a bank account with methods to deposit and withdraw money. Include a method to display the balance.

(3) Write a class for a `Person` that has a name and age. Include a method to display the person's details.

(4) Write a class to represent a `Car` with attributes such as make, model, and year. Include a method to display the car's information.

(5) Write a class for a `Circle` with a radius. Include methods to calculate the circumference and area.

(6) Write a class to represent a simple `Student` with attributes like name and roll number. Include a method to display student details.

(7) Write a class `Book` with attributes such as title, author, and publication year. Include a method to display book details.

(8) Write a class `Triangle` with attributes for the lengths of the sides. Include a method to calculate the area using Heron's formula.

(9) Write a class `Date` with attributes for day, month, and year. Include a method to display the date in a readable format.

(10) Write a class `Point` that represents a point in 2D space with x and y coordinates. Include methods to calculate the distance from another point.

(11) Write a class `Complex` to represent complex numbers with real and imaginary parts. Include methods to add and subtract complex numbers.

(12) Write a class `Stack` that implements a simple stack with push and pop operations.

(13) Write a class `Queue` that implements a simple queue with enqueue and dequeue operations.

(14) Write a class `Employee` with attributes for ID, name, and salary. Include a method to display employee details.

(15) Write a class `Matrix` with methods to perform matrix addition and multiplication.
