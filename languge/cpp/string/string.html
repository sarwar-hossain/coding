Understanding Strings in C++

Easy Problems

(1) Write a program to declare a string, initialize it with a value, and print it.

(2) Write a program to find the length of a given string.

(3) Write a program to concatenate two strings.

(4) Write a program to compare two strings and check if they are equal.

(5) Write a program to find a substring within a string.

(6) Write a program to replace all occurrences of a character in a string.

(7) Write a program to convert a string to uppercase.

(8) Write a program to convert a string to lowercase.

(9) Write a program to reverse a given string.

(10) Write a program to check if a string is a palindrome.

(11) Write a program to count the number of vowels in a string.

(12) Write a program to find the first occurrence of a character in a string.

(13) Write a program to find the last occurrence of a character in a string.

(14) Write a program to split a string into words based on spaces.

(15) Write a program to trim leading and trailing spaces from a string.
