Understanding Variables in C++

Easy Problems

(1) Declare an integer variable, assign a value to it, and print the value.

(2) Declare a float variable, assign a value, and print it with 2 decimal places.

(3) Declare a char variable, assign a character to it, and print the character.

(4) Declare a string variable, assign a string value to it, and print the string.

(5) Declare an integer variable, increment its value by 10, and print the result.

(6) Declare two integer variables, assign values to them, and print their sum.

(7) Declare a variable of type double, assign a value, and print it in scientific notation.

(8) Declare a boolean variable, assign a value, and print its value as true or false.

(9) Declare a variable of type long, assign a large value, and print the value.

(10) Declare a variable of type unsigned int, assign a value, and print it.

(11) Declare a constant variable, assign a value to it, and print the value. Explain why the value cannot be changed.

(12) Use a variable to store the result of a division operation and print the result as a floating-point number.

(13) Declare two variables of different types (e.g., int and float), perform an operation between them, and print the result.

(14) Declare a variable of type auto and assign it a value. Print the type of the variable using typeid.

(15) Declare an array of integers, assign values to the elements, and print all the elements.
