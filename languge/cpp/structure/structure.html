Understanding Structures in C++

Easy Problems

(1) Define a structure to represent a point in 2D space with `x` and `y` coordinates. Write a program to create a point and display its coordinates.

(2) Define a structure to represent a rectangle with `length` and `width`. Write a program to calculate and display the area of the rectangle.

(3) Define a structure to represent a student with `name`, `rollNumber`, and `marks`. Write a program to input and display student details.

(4) Define a structure to represent a book with `title`, `author`, and `price`. Write a program to input and display book details.

(5) Define a structure to represent a date with `day`, `month`, and `year`. Write a program to input and display the date.

(6) Define a structure to represent a circle with `radius` and `center` (a point). Write a program to calculate and display the circumference of the circle.

(7) Define a structure to represent a complex number with `real` and `imaginary` parts. Write a program to add two complex numbers.

(8) Define a structure to represent a car with `make`, `model`, and `year`. Write a program to compare two cars and display which one is newer.

(9) Define a structure to represent a triangle with `side1`, `side2`, and `side3`. Write a program to calculate and display the perimeter of the triangle.

(10) Define a structure to represent a person with `name`, `age`, and `height`. Write a program to input and display the details of a person.

(11) Define a structure to represent a bank account with `accountNumber`, `accountHolder`, and `balance`. Write a program to deposit and withdraw money from the account.

(12) Define a structure to represent a student with `name`, `id`, and an array of `grades`. Write a program to calculate the average grade of the student.

(13) Define a structure to represent a product with `productID`, `name`, and `price`. Write a program to find and display the most expensive product from a list.

(14) Define a structure to represent a time with `hours`, `minutes`, and `seconds`. Write a program to convert the time to seconds.

(15) Define a structure to represent a 3D point with `x`, `y`, and `z` coordinates. Write a program to calculate the distance between two 3D points.
