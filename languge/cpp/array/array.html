Understanding Arrays in C++

Easy Problems

(1) Write a program to declare an array of integers, initialize it with values, and print its elements.

(2) Write a program to find the sum of all elements in an array of integers.

(3) Write a program to find the maximum and minimum values in an array of integers.

(4) Write a program to reverse the elements of an array.

(5) Write a program to count the number of occurrences of a specific value in an array.

(6) Write a program to sort an array of integers in ascending order using a simple sorting algorithm.

(7) Write a program to merge two sorted arrays into a single sorted array.

(8) Write a program to remove duplicate elements from an array.

(9) Write a program to find the average of elements in an array of floating-point numbers.

(10) Write a program to rotate the elements of an array to the right by a specified number of positions.

(11) Write a program to find the second largest element in an array.

(12) Write a program to check if an array is a palindrome.

(13) Write a program to compute the product of all elements in an array of integers.

(14) Write a program to find the frequency of each element in an array.

(15) Write a program to find the common elements between two arrays.
