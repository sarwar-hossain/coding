Understanding Data Types in C++

Easy Problems

(1) Declare a variable of type int, assign a value to it, and print the value.

(2) Declare a variable of type float, assign a value to it, and print the value.

(3) Declare a variable of type double, assign a value to it, and print the value.

(4) Declare a variable of type char, assign a character to it, and print the character.

(5) Declare a variable of type bool, assign true or false, and print the value.

(6) Declare a variable of type long, assign a large integer value to it, and print the value.

(7) Declare a variable of type unsigned int, assign a positive integer value, and print the value.

(8) Declare a variable of type short, assign a small integer value, and print the value.

(9) Declare a variable of type wchar_t, assign a wide character, and print the character.

(10) Declare a variable of type long long, assign a very large integer value, and print the value.

(11) Declare a variable of type auto, assign a value, and print the value. Explain how the type is deduced.

(12) Declare an array of integers and initialize it with values. Print all elements.

(13) Declare an array of floats and initialize it with values. Print all elements.

(14) Declare a pointer of type int, assign it the address of an integer variable, and print the value pointed to by the pointer.

(15) Declare a constant variable of type double and print its value. Explain why it cannot be modified.
