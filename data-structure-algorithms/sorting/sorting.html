Sorting and Searching Problems

Easy Problems

(1) Implement Bubble Sort.

(2) Implement Selection Sort.

(3) Implement Insertion Sort.

(4) Search an element in a sorted array using Binary Search.

(5) Find the first and last occurrence of an element in a sorted array.

(6) Implement Merge Sort.

(7) Implement Quick Sort.

(8) Implement Linear Search.

(9) Implement Counting Sort.

(10) Implement Radix Sort.

(11) Implement Heap Sort.

(12) Find the kth smallest element in an unsorted array.

(13) Find the kth largest element in an unsorted array.

(14) Search an element in a rotated sorted array.

(15) Count occurrences of an element in a sorted array.

(16) Find the peak element in an array.

(17) Implement Shell Sort.

(18) Implement Tim Sort.

(19) Find the maximum and minimum element in an array using the minimum number of comparisons.

(20) Implement Binary Insertion Sort.

Medium Problems

(21) Implement 3-way Quick Sort.

(22) Find the median of two sorted arrays.

(23) Find the intersection of two arrays.

(24) Search in a 2D matrix where rows and columns are sorted.

(25) Implement an algorithm to search an element in a nearly sorted array.

(26) Implement a binary search for a given range of numbers (lower bound and upper bound).

(27) Implement a binary search for a circularly sorted array.

(28) Find the minimum element in a sorted and rotated array.

(29) Count the number of inversions in an array.

(30) Find the smallest missing positive number in an unsorted array.

(31) Implement a ternary search.

(32) Implement a jump search.

(33) Implement exponential search.

(34) Implement Fibonacci search.

(35) Search for an element in a bitonic array.

(36) Find the smallest element in a rotated sorted array with duplicates.

(37) Implement a binary search for a floating-point number.

(38) Implement Kth smallest/largest element in an unsorted array using Quickselect.

(39) Implement counting inversions using merge sort.

(40) Implement a bucket sort.

Hard Problems

(41) Implement a search for the largest rectangle of 1s in a binary matrix.

(42) Implement a search for the maximum sum rectangle in a 2D matrix.

(43) Implement an algorithm to search for the maximum length of a subarray with a given sum.

(44) Implement an algorithm to search for the smallest subarray with a sum greater than a given value.

(45) Implement an algorithm to find the k-th smallest element in a matrix.

(46) Implement a sorting algorithm with a time complexity better than O(n log n).

(47) Implement an algorithm to find the longest consecutive subsequence in an unsorted array.

(48) Implement an algorithm to merge overlapping intervals in an array.

(49) Implement an algorithm to find the maximum product of a subarray.

(50) Implement an algorithm to search for the median of a stream of integers.
